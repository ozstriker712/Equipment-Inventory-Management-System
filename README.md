# Equipment-Inventory-Management-System
A simple and efficient inventory management program developed with C# (.NET) using Entity Framework 6 and SQL Server as the database.
The system helps manage and organize equipment across departments, allowing users to add, update, and delete items easily through a modern DevExpress-based interface.

The database includes two SQL triggers for insert/update operations and another for delete operations — ensuring data integrity and tracking all changes.
It also includes a permission system per user, allowing administrators to control what each user can access or modify.

Features:

Developed with C# .NET, Entity Framework 6, and SQL Server.

Beautiful and smooth DevExpress interface.

Add, edit, and delete equipment easily.

SQL triggers for data control and action tracking.

User permission system for fine-grained access control.

Organized, lightweight, and easy to use for any organization.

اسم المشروع: نظام جرد العتاد وإدارة المعدات

الوصف:
برنامج بسيط وفعّال تم تطويره بلغة C# .NET باستخدام Entity Framework 6 وقاعدة بيانات SQL Server.
يساعد النظام في إدارة وتنظيم العتاد داخل الإدارات، مع إمكانية الإدخال، التعديل، والحذف من خلال واجهة عصرية مبنية بمكتبة DevExpress.

تحتوي قاعدة البيانات على تريجرين (Triggers) لعمليات الإدخال والتعديل، وتريجر آخر للحذف لضمان سلامة البيانات وتسجيل العمليات.
كما يحتوي البرنامج على نظام صلاحيات لكل مستخدم، يتيح التحكم بما يمكن لكل مستخدم الوصول إليه أو تعديله.

الميزات:

مطور بـ C# .NET و Entity Framework 6 و SQL Server.

واجهة جميلة وسلسة من DevExpress.

إمكانية إدخال، تعديل، وحذف المعدات بسهولة.

استخدام تريجرات SQL لمتابعة التغييرات وضمان سلامة البيانات.

نظام صلاحيات لكل مستخدم للتحكم الدقيق في الوصول.

برنامج منظم، خفيف، وسهل الاستخدام في أي إدارة أو مؤسسة.
